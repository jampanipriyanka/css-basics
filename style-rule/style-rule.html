Style Rule

these are the rules that we need to remember while writing styles for our web pages(html document)

        /* Here we call "p" as - selector  */
        /* Here we call "color" as -property */
        /* Here we call "red" as value to "color" property */
        /* Here we call "color:red;" as declaration */
        /* Here we call "p { .... }" as rule */
          p{
            color: red;
            background-color:White;
          }  
          div{
            font-size: 28px;
          }
